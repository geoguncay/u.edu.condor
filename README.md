# Proyecto Unidad educativa Cordillera del Condor

Pagina Web para la Unida Educativa

## Grupo de desarrollo

## Frameworks utilizados

- React
- NodeJS(16.15.1)
- Tailwindcss
- Gestor de paquetes npm

## Comandos

### Front-end

```shell
cd u.edu.condor/
```

```shell

npm install
npm start

```

### Back-end

```shell
cd web-app/
cd src/
cd server/
node server.js
```
